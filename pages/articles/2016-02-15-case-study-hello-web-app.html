I'm a visual person with a background in building HTML/CSS websites, and programming never felt real unless I was seeing results in the browser. However, programming tutorials all tended to show things working in the command line and left the website stuff (templates and whatnot) until the end of the tutorial, or not at all. In 2014 I started working on my "Django for Designers" book idea, which evolved into "Hello Web App." ## The books **Website: [http://hellowebapp.com](http://hellowebapp.com)** ![](http://static1.squarespace.com/static/543c1b40e4b0f23a04303889/543f4799e4b033d6ec5036ed/56c24c3004426207ff83271d/1455574155595/2.jpg) ![](http://static1.squarespace.com/static/543c1b40e4b0f23a04303889/543f4799e4b033d6ec5036ed/56c24c82f8baf36bfdacca83/1455574189639/hellowebapp3.jpeg) ![](http://static1.squarespace.com/static/543c1b40e4b0f23a04303889/543f4799e4b033d6ec5036ed/56c24c60f8baf36bfdacc94d/1455574160335/IMG_6466.jpg) **_Hello Web App_ walks readers through the process of building their first web app. **I start with templates and the "website stuff" first, building the website while introducing concepts like models and the database slowly but surely. _Hello Web App_ is 144 pages and concludes with launching a basic but usable web app on Heroku. **_Hello Web App: Intermediate Concepts_ is for anyone who has a basic web app and wants to build more complex projects** like adding payments with Stripe, allowing for user-uploaded images, adding an API, and more. As compared to _Hello Web App_, the chapters in _Hello Web App: Intermediate Concepts _are not chronological and can be tackled individually. _Hello Web App_ launched May 4th 2015 and _Hello Web App: Intermediate Concepts_ launched on December 7th 2015. ## Kickstarter campaigns For each project I ran a Kickstarter campaign: [Hello Web App raised $12,136 with 540 backers](https://www.kickstarter.com/projects/1868398473/hello-web-app-intro-to-building-web-apps-with-djan), ending in May 2014. ...and [Hello Web App: Intermediate Concepts raised $13,365 with 319 backers](https://www.kickstarter.com/projects/1868398473/hello-web-app-intermediate-concepts), ending in October 2015. **While I got an offer from a publisher for _Hello Web App_, I decided to self-publish.** This way, I could keep 100% of the royalties, in exchange for being on the hook for the design and marketing. Thankfully, editorial design was one of my favorite classes at university.
