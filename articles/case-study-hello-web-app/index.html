Case study: Hello Web App

(I'm moving to Toronto and looking for a job (!) and thought I'd deep dive into my two main projects to showcase my work.)

In 2011, I taught myself how to code Python and Django to launch my startup, WeddingLovely. I got something working, but over the years of working with Python and Django, I became more and more frustrated about how programming in general was taught.

I'm a visual person with a background in building HTML/CSS websites, and programming never felt real unless I was seeing results in the browser. However, programming tutorials all tended to show things working in the command line and left the website stuff (templates and whatnot) until the end of the tutorial, or not at all.

In 2014 I started working on my "Django for Designers" book idea, which evolved into "Hello Web App."

The books

Website: http://hellowebapp.com

Hello Web App walks readers through the process of building their first web app. I start with templates and the "website stuff" first, building the website while introducing concepts like models and the database slowly but surely. Hello Web App is 144 pages and concludes with launching a basic but usable web app on Heroku.

Hello Web App: Intermediate Concepts is for anyone who has a basic web app and wants to build more complex projects like adding payments with Stripe, allowing for user-uploaded images, adding an API, and more. As compared to Hello Web App, the chapters in Hello Web App: Intermediate Concepts are not chronological and can be tackled individually.

Hello Web App launched May 4th 2015 and Hello Web App: Intermediate Concepts launched on December 7th 2015.

Kickstarter campaigns

For each project I ran a Kickstarter campaign:

Hello Web App raised $12,136 with 540 backers, ending in May 2014.

...and Hello Web App: Intermediate Concepts raised $13,365 with 319 backers, ending in October 2015.

While I got an offer from a publisher for Hello Web App, I decided to self-publish. This way, I could keep 100% of the royalties, in exchange for being on the hook for the design and marketing. Thankfully, editorial design was one of my favorite classes at university.

The Kickstarter campaigns were crucial — I could use the campaigns to build excitement about the books, sell pre-orders, and (almost most importantly) raise money to cover the costs of printing 1,000 paperbacks.

Workshops

The content in Hello Web App translates very well to workshops. I've given workshops at PyCon 2014, DjangoCon EU 2015, DjangoCon US 2015, and PyTennessee 2016, and will be giving it at PyCon US 2016 as well.

I've also started giving non-conference workshops, like the Hackbright + Hello Web App workshop that I gave in September 2015 (photos shown above). We had around 75 attendees!

The website

While this blog is still (much to my chagrin) on Squarespace, I recently designed and built a new Hello Web App website from scratch, using the Cactus static site builder.

I'm particularly proud of my blog pages.

Last thoughts

Writing, producing, and marketing both Hello Web App books has been one of the best things I've ever done. I've been able to make a measurable difference, teaching thousands of people how to code and launch their own web apps.

I like writing books so much that I'm pretty set on writing Hello Web Design, teaching design principles aimed at non-designers and programmers.
